# OClang
OCaml bindings to libclang.

### Building and installation

    $ ./autogen.sh
    $ ./configure
    $ make

For installation with ocamlfind:

    $ make install-findlib

Otherwise:

    $ make install

For documentation generation:

    $ make documentation
